# hugo-theme-dream-ink

这是一款基于 [hugo-theme-dream](https://github.com/g1eny0ung/hugo-theme-dream) 的主题，我用它作为我 [个人博客](http://ryunpu.github.io/) 的主题，喜欢的同学可以折腾下。

主题新增了以下特性：

* 展示首页
* 站内搜索
* 图片浏览
* UI 及其他
